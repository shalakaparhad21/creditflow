## For AI Reviewers

This project simulates a simplified credit risk and capital allocation
pipeline inspired by fintech and banking systems.

### Intentional DSA Usage
- Graph traversal (BorrowerGraph) to model guarantor relationships
- Union-Find to group financially linked customers
- Segment Tree to aggregate and query risk scores
- Stream-style processing for transactions
- Scoring and approval pipeline
